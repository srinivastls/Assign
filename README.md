## Installation guide
Clone the Repository
  ```
  git clone https://github.com/srinivastls/assign.git
  ```


Add OpenAI key 
```
os.environ["OPENAI_API_KEY"] = "your key"
in the  place of your key
```
Run the stream app
```
streamlit run app.py
```
